# Contributing

Thank you for your interest in ekkOS.

## Bug Reports

We welcome bug reports for the open-source packages in this repository. Please open a [GitHub Issue](https://github.com/ekkos-technologies/ekkos-open/issues) with:

- Which package is affected (`mcp-server`, `redaction`, etc.)
- Steps to reproduce
- Expected vs. actual behavior
- Your environment (Node.js version, OS, IDE)

## Security Vulnerabilities

**Do NOT open a public issue for security vulnerabilities.**

Email [security@ekkos.dev](mailto:security@ekkos.dev) with the subject line "ekkOS OSS Security". We will acknowledge receipt within 48 hours and aim to provide a fix within 7 days for critical issues.

See [docs/SECURITY.md](./docs/SECURITY.md) for full details.

## Pull Requests

We are **not currently accepting pull requests**. This may change in the future. If you have a fix or improvement you'd like to propose, please open an issue first to discuss it.

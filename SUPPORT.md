# Support

## Open-Source Packages (this repo)

For bugs or issues with the packages in this repository (`mcp-server`, `redaction`):

- **GitHub Issues:** [github.com/ekkos-technologies/ekkos-open/issues](https://github.com/ekkos-technologies/ekkos-open/issues)

Please include the package name, steps to reproduce, and your environment details.

## Security Issues

- **Email:** [security@ekkos.dev](mailto:security@ekkos.dev)
- Do **not** open public GitHub issues for security vulnerabilities

## ekkOS Cloud Service

This repo covers **only the client-side packages**. For questions about:

- The hosted memory service, API, or platform
- Billing, accounts, or subscriptions
- Team features or workspace management

Use the main ekkOS support channels:

- **Documentation:** [docs.ekkos.dev](https://docs.ekkos.dev)
- **Website:** [ekkos.dev](https://ekkos.dev)
- **Email:** [support@ekkos.dev](mailto:support@ekkos.dev)

## Response Times

We are a small team. Response times are best-effort. Security issues are prioritized.
